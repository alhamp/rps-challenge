Intructions

Write a program where you can compete with the computer in **Rock, Paper, Scissors**. You should be able to run this code in your Google Chrome console and have it work!

About

Rock-paper-scissors is a hand game usually played by two people, where players simultaneously form one of three shapes with an outstretched hand. The "rock" beats "scissors", the "scissors" beats "paper" and the "paper" beats "rock"; if both players throw the same shape, the game is tied.

Getting Started. Creating your game.

You will be creating a digital version of the rock, paper, scissors game for the browser. Users should at least be able to see the results in the console. What this project already has included:index.html, styles.css, images, rock-paper-scissors.js, README.md

First Level Requirements. Do the following first.

  1. A game consists one round.
  2. "Let's Play Rock, Paper, Scissors!" should appear in the console.
  3. The player is prompted to choose a selection by typing "rock" , "paper", or "scissors".
  4. The computer opponent randomly chooses between rock, paper, and scissors.
  5. Output "you" and "computer" player choices. Examples: a) "You chose rock" b) "Computer chose scissors"
  6. Output the result of the game before exiting. Examples: a) "You won, congrats champion!" b) "Computer wins, now the robots take over!" c) "It is a Tie! Battle to the death!"
  7. If the player enters an invalid option, print an error message.
  8. If both players choose the same option, it is a tie.

Second Level Requirements. Challenge Accepted?

  1. If you completed everything from First Level Requirements.
  2. Have what is shown in the console from First Level now show on the browser page.
  3. The game ends when the player or the computer wins the best out of three rounds played.
  4. Ties do not count as a win.
  5. Output the results of each round. Examples: a) "You chose scissors." b) "Computer chose scissors." c) "This round was a tie. Try Again!"
  6. Output the winner of all three rounds Examples: a) "Computer wins the game!"
  7. Total Output the result of the game before exiting the game. Examples: a) "Your scores: tie 1 won 2 loses 0!" b) "Computer scores: ties 1 wins 1 loses 1!"

Boss Battle Requirements. Bring it!

  1. Create a score board on browser page.
  2. Have what is shown in the console from First Level now show on the browser page.
  3. Make the game prompt show up only when clicking the "Play Now!" button.
